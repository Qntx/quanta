    API 接口文档¶

    Class Data¶

    Data 是 ΣData 的本地数据管理核心，主要负责存储和操作本地 OHLCV 数据。它以高效的 NumPy 数据存储为基础，支持快速数据操作。

    核心功能¶

    功能 描述
    初始化数据 使用 Pandas DataFrame、NumPy 数组或列表初始化数据。
    添加数据 动态添加单条新的 OHLCV 数据。（动态添加多条数据支持正在开发中）
    更新数据 更新最新的 OHLCV 数据（用于修改最后一条记录）。
    删除数据 删除最旧的 OHLCV 数据或清空所有数据。
    事件回调 支持注册同步与异步事件回调（如数据添加、更新、删除等事件）。

    外部接口¶

    外部方法（Public Methods）¶

    1. 初始化数据：init¶
    def init(
    self,
    data: Union[pd.DataFrame, np.ndarray, List[List[Any]]],
    call_callbacks: bool = True,
)

    功能说明：使用 Pandas DataFrame、NumPy 数组或列表初始化数据。

    参数：

    参数 类型 描述
    data pd.DataFrame / np.ndarray / list 初始化的数据，必须包含以下列顺序：[timestamp, open, high, low, close, volume]。
    call_callbacks bool（默认：True） 是否在初始化时触发已注册的回调事件。

    异常：

    • ValueError：如果数据格式错误或超出存储容量。
    • TypeError：如果输入的数据类型不被支持。
    2. 添加数据：add¶
    def add(
    self,
    data: Union[pd.Series, pd.DataFrame, np.ndarray, List[Any]],
    call_callbacks: bool = True,
)

    功能说明：添加单条或多条新的 OHLCV 数据。此方法会确保容量足够，如容量不足，将自动移除最旧的数据点。

    参数：

    参数 类型 描述
    data pd.Series / pd.DataFrame / np.ndarray / list 添加的数据。
    call_callbacks bool（默认：True） 是否触发添加回调事件。

    异常：

    • ValueError：如果输入数据格式不正确。
    • RuntimeError：如果添加操作在内部出错。

    Warning

    当前仅支持添加单条 OHLCV 数据。

    3. 更新数据：update¶
    def update(
    self,
    data: Union[pd.Series, np.ndarray, List[Any]],
    call_callbacks: bool = True,
)

    功能说明：更新最新的一条 OHLCV 数据。该方法会替换当前数据集中最后一条记录。

    参数：

    参数 类型 描述
    data pd.Series / np.ndarray / list 更新的数据，必须与当前最新数据匹配。
    call_callbacks bool（默认：True） 是否触发更新回调事件。

    异常：

    • IndexError：当数据集中没有数据时无法更新。
    • ValueError：如果输入数据不符合格式。
    4. 删除数据：remove¶
    def remove(
    self, 
    call_callbacks: bool = True
)

    功能说明：删除数据集中最新的一条记录。如果当前数据集中没有数据，则抛出异常。

    参数：

    参数 类型 描述
    call_callbacks bool（默认：True） 是否触发删除回调事件。

    异常：

    • IndexError：当数据集中没有记录时无法删除。
    5. 清空数据：clear¶
    def clear(
    self, 
    count: Optional[int] = None, 
    call_callbacks: bool = True
)

    功能说明：清空数据集中指定数量的记录。如果未指定 count 参数，将删除所有记录。

    参数：

    参数 类型 描述
    count int（可选） 要清空的记录数量。如果为 None，则清空所有数据。
    call_callbacks bool（默认：True） 是否在清空数据后触发回调事件。

    异常：

    • ValueError：如果 count 为负值或不合法。
    • 无数据时触发回调，但无实际异常。

    外部属性（Public Properties）¶

    1. timestamp¶
    @property
def timestamp(self) -> np.ndarray
    • 描述: 返回 timestamp 列的 NumPy 只读数组，包含数据的时间戳信息。
    • 类型: np.ndarray
    2. open¶
    @property
def open(self) -> np.ndarray
    • 描述: 返回 open 列的 NumPy 只读数组，表示开盘价。
    • 类型: np.ndarray
    3. high¶
    @property
def high(self) -> np.ndarray
    • 描述: 返回 high 列的 NumPy 只读数组，表示最高价。
    • 类型: np.ndarray
    4. low¶
    @property
def low(self) -> np.ndarray
    • 描述: 返回 low 列的 NumPy 只读数组，表示最低价。
    • 类型: np.ndarray
    5. close¶
    @property
def close(self) -> np.ndarray
    • 描述: 返回 close 列的 NumPy 只读数组，表示收盘价。
    • 类型: np.ndarray
    6. volume¶
    @property
def volume(self) -> np.ndarray
    • 描述: 返回 volume 列的 NumPy 只读数组，表示交易量。
    • 类型: np.ndarray
    7. np¶
    @property
def np(self) -> np.ndarray
    • 描述: 返回整个 OHLCV 数据集的 NumPy 只读视图，包含所有列 (timestamp, open, high, low, close, volume)。
    • 类型: np.ndarray
    8. df¶
    @property
def df(self) -> pd.DataFrame
    • 描述: 返回整个 OHLCV 数据集的 Pandas DataFrame 格式，便于数据操作和分析。
    • 类型: pd.DataFrame
    9. size¶
    @property
def size(self) -> int:
    • 描述: 返回当前存储的 OHLCV 数据点的数量。
    • 类型: int
    10. capacity¶
    @property
def capacity(self) -> int:
    • 描述: 返回数据实例的最大存储容量。
    • 类型: int
    11. delete_size¶
    @property
def delete_size(self) -> int:
    • 描述: 返回数据实例已满时删除的数据数量。
    • 类型: int
    12. columns¶
    @property
def columns(self) -> List[str]:
    • 描述: 返回数据集的列名，默认为 ["timestamp", "open", "high", "low", "close", "volume"]。
    • 类型: List[str]
